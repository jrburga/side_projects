Erebus

===============


Entry in PyWeek #16, Theme "Nemesis"
Team: Paralax

Members: Jake Burga (Ryan42)



DEPENDENCIES: 
	

You might need to install some of these before running the game:

  
		Python:     http://www.python.org/
  
		PyGame:     http://www.pygame.org/
  

RUNNING THE GAME: 
	

On Windows or Mac OS X, locate the "run_game.pyw" file and double-click it.

 Otherwise open a terminal / console and "cd" to the game directory and run:

  
	python run_game.py





HOW TO PLAY THE GAME:
	Use the arrow keys to move through the star field. Avoid getting pulled in by your *nemesis*.
